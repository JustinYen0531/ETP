# ETP Game Show - 互動式學術大富翁設計文件

## 1. 遊戲概述 (Project Overview)
這是一個基於 Web 的英語互動競賽遊戲，模仿大富翁（Monopoly）的機制，結合大學四大專業學科（會計、統計、微積分、經濟）的內容進行答題競賽。

## 2. 遊戲配置 (Game Configuration)
- **玩家數量**: 建議 4 組玩家/團隊。
- **學科主題**:
  1. **Accounting (會計)**
  2. **Statistics (統計)**
  3. **Calculus (微積分)**
  4. **Economics (經濟)**
  5. **Life (生活)**
  6. **Challenge (挑戰)**
- **地圖格子 (Tiles)**: 共 10 格（由 0-9 索引）-> **特性：題目不重複**
  - `0`:  Life/Challenge Tile (繞一圈獲得50分)
  - `1, 2`: Accounting Tiles 
  - `3, 4`: Statistics Tiles
  - `5, 6`: Calculus Tiles
  - `7, 8`: Economics Tiles
  - `9`: Life/Challenge Tile

## 3. 題目機制 (Question Banks)
- **等級劃分**：簡化為三個等級，每個科目、每個格子皆有 3 題（Level 1, 2, 3 各一題）。
- **等級與題目類型**:
  - **Level 1: 填充題** (名詞填充)
  - **Level 2: 計算題** (涉及簡易運算)
  - **Level 3: 開放性問題** (需回答正確邏輯，無絕對標準答案)
- **總題數**: 2 (會計格子) * 3 (等級) + 2 (統計) * 3 + ... = 8 格 * 3 題 = 24 題 (不含生活題)。

## 4. 核心遊戲流程 (Core Gameplay Loop)
### A. 移動 (Movement)
- 玩家擲骰子並根據步數移動。
- 如果落點格子已達到「滿級 (Level 3)」，該格子將被直接略過。

### B. 挑戰與房產 (Property Mechanics)
1. **無人佔領時**:
   - 玩家必須回答 **Level 1** 問題。
   - **答對**: 獲得該格房產（Level 1）。
   - **答錯**: 所有權維持無主。
2. **他人佔領時**:
   - 玩家必須回答該格的 **當前等級 + 1** 的問題。
   - **答對 (奪取成功)**: 該格所有權轉移給當前玩家，且等級增加。
   - **答錯 (防禦成功)**: 所有權維持原狀，且該房產「自動升一級」。

## 5. 遊戲結束條件 (End Conditions)
1. 全部的題目皆已回答完畢。
2. 全部的格子皆已達到滿級（或被「鎖定」）。
3. 統計每個組別擁有的房產等級總分數，最高者獲勝。

## 6. 技術棧建議 (Tech Stack)
- **Frontend**: HTML5, CSS3 (Glassmorphism 風格), JavaScript.
- **Visuals**: 動態骰子動畫、平滑的卡片翻轉、得分統計榜。
